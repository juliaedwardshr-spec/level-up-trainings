LEVEL UP TRAININGS V12 INSTALLABLE APP

This version turns the site into an installable Progressive Web App (PWA).

Includes:
- manifest.webmanifest
- service-worker.js
- app icons
- install button on the homepage
- offline caching for the main pages
- same Netlify drag-and-drop workflow

Deploy:
1. Unzip this folder.
2. Confirm these files are at the top level:
   index.html, about.html, services.html, quest.html, accessibility.html, contact.html,
   styles.css, app.js, mascot.png, manifest.webmanifest, service-worker.js,
   icon-192.png, icon-512.png
3. Drag the whole folder into Netlify Production Deploys.

Install:
- On iPhone/Safari: open the site, tap Share, then Add to Home Screen.
- On Android/Chrome: tap Install App or Add to Home Screen.
- On desktop Chrome/Edge: use the install icon in the address bar or the Install App button.
