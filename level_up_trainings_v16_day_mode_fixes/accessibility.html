Accessibility Commitment

Training should not leave people behind.

Level Up Trainings supports plain-language structure, readable layouts, screen-reader friendly formatting, and accessibility-minded learning design.

The Braille-friendly question list is written as plain text so users with refreshable Braille displays or screen readers can access the intake questions more easily.

Accessible design can include

  • Plain-language rewrites
  • Better content hierarchy
  • Readable contrast and spacing
  • Text alternatives and simplified instructions
